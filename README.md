# sikarwar.me — AirCnC

A personal website built as an Airbnb-style listing page for Saurav's Golden Retriever. Hosted at [sikarwar.me](https://sikarwar.me).

## About

AirCnC lets visitors browse details about the dog, view photos, check care & schedule info, read reviews, and send a booking request directly via email.

## Stack

- Plain HTML, CSS, and vanilla JavaScript — no frameworks or build tools
- Fully static, no backend required

## Structure

```
sikarwar.me/
├── index.html        # Main page
├── aircnc.css        # Styles
├── aircnc.js         # UI interactions (auth modal, nav drawer, gallery)
└── images/           # Photos and icons
```

## Features

- Hero image gallery with full-screen overlay
- Sticky booking card with date picker and price breakdown
- Sign in / Sign up modal (UI only)
- Mobile-responsive nav drawer
- Care schedule, activities, reviews, and location sections
